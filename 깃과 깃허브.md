# 깃과 깃허브

## 깃 : 컴퓨터에 기록되는 모든 파일들의 변화를 기록한다, 각기 다른 버전의 모든 파일들을 저장할 수 있다.

그렇기 때문에 다른 형식의 파일들 중 중심에 있는 한가지 파일을 바꿀 수 있고 모든 버전의 파일을 수정 가능하다.

## 깃허브 : 깃 클라우드 같은 개념, 에디터가 만든 깃 파일을 다른 에디터가 볼 수 있다.

그렇기 때문에 팀 프로젝트를 하며 깃 파일을 서로 수정하거나 함께 사용하기 유용하며 여러 버전의 깃 파일을 공유하여 사용할 수 있다.

# 비유

#### 깃허브 : 쿠팡 물류센터

#### 깃  : 택배
