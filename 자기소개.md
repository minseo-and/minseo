``` c++
#include <stdio.h>
int main()
{
	int a, i, j;
	a=10000;
	for(i=0; i<a; i++){
		for(j=0; j<10000; j++)
		printf("곱셈을 하겠습니다 %d*%d=%d", i, j, i*j);
	}	
}
```

#### *WHY*
